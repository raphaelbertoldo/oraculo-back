## Run 
 To run this project, first install the dependencies:
  ```bash 
  npm install
  ```

Next, start the project:
```bash 
npm run start:dev
```

### Build
To build the application for production:

```bash

# or

```

  

To start the production server:

  

```bash

# or

```
